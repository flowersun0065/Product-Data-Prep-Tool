📖 三种使用方式
================================================================================

方式 A - 命令行脚本（推荐）⭐⭐⭐
  最快、最便宜、最灵活
  python product_cleaner.py --input 你的文件.xlsx

方式 B - Web 应用 ⭐⭐
  可视化界面、友好易用
  python product_cleaner_web.py
  然后访问 http://localhost:5000

方式 C - 分阶段处理 ⭐⭐
  先测试小数据集，再处理全量
  python product_cleaner.py --input data.xlsx --batch-size 100
  （测试前 1000 行）
